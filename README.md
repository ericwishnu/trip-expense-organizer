# Holiday Expense Tracker

A Laravel + Filament app for managing trips and tracking expenses by day.

## Features
- User registration and login
- Trips with dates, destination, and budget
- Day entries (Day 1, Day 2, etc.) per trip
- Expenses tied to a specific trip day

## Setup
- Ensure your database is configured in .env (SQLite is configured by default).
- Run migrations with Artisan.
- Start the Laravel development server.
- Open the Filament panel at /admin and register a new user.

## Usage
1. Create a trip.
2. Add trip days (e.g., Day 1, Day 2).
3. Add expenses for each day.
